# 🍽️ Hotel Billing System in C

## 📌 Overview

The Hotel Billing System is a console-based C application that allows users to select food categories, choose menu items, enter quantities, and automatically calculates the total bill.

This project was developed to practice:
- Nested if-else statements
- User input handling
- Conditional programming
- Basic billing system logic

---

## ✨ Features

- Separate Veg, Non-Veg, and Drinks menus
- Menu-driven interface
- Quantity selection
- Automatic bill calculation
- Invalid input handling

---

## 🛠️ Technologies Used

- C Programming
- Turbo C / GCC
- VS Code

---

## 📋 Menu Categories

### 🥗 Veg
- Paneer Butter Masala
- Veg Biryani
- Dal Fry

### 🍗 Non-Veg
- Chicken Biryani
- Butter Chicken
- Chicken Curry

### ☕ Drinks
- Tea
- Coffee
- Cold Drink

---

## ▶️ How to Run

1. Compile the program

```bash
gcc hotel_billing.c -o hotel
```

2. Run the executable

```bash
./hotel
```
